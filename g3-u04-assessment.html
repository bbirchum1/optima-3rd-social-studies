Part B: Portfolio Response
Questions 5–6 · 12 points
5Written Response ★ 6 pts
Think of a question a map could help you answer — for example: “Which region gets the most rain?” or “How far is it from Florida to Texas?” Write the question, then name which type of map would best answer it and explain why.
Write 3–5 sentences. Name the map type and explain your thinking.
6Written Response ★ 6 pts
If an explorer from the 1500s drew a map of a coastline, that map is a primary source. Explain why, and describe one thing we could learn from it that a modern textbook might leave out.
Write 3–5 sentences. Explain what makes it a primary source and what we can learn.

Question 7 · 4 points
7Virtue Reflection ★ 4 pts

Write 2–4 sentences using the word “honesty” in your answer.
